## Windows 11/ Windows Server 2022

在 Demo 版期间，组员 W 在他的宁美魔方小主机上测试时发现视频播放缓慢，于是在正式版我们开发了硬件解码功能，但是为保证，该功能位于 “设置->实验选项->硬解” ，该功能理论上无需重启生效，但是为了保证生效推荐重启游戏（关闭重开）
